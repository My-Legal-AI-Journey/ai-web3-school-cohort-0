# AI × Web3 School — 个人学习仓库

Personal learning journal and proof-of-work for [AI × Web3 School](https://aiweb3.school/zh/handbook/).

## 固定入口

| 资源 | 链接 |
|------|------|
| Handbook | https://aiweb3.school/zh/handbook/ |
| Learning Agent 启动 Prompt | https://aiweb3.school/learning-agent.zh.txt |
| WCB 训练营 | https://web3career.build/zh/programs/AI-Web3-School |
| WCB Learning | https://web3career.build/zh/programs/AI-Web3-School#tab=learning |
| WCB Agent API | https://web3career.build/llms.txt |

## 本仓库用途

- 学习计划与每日打卡记录（`learning-plan.md`、`daily/`）
- Week 任务与实验沉淀（`tasks/`、`experiments/`）
- Handbook 问题与改进建议（`handbook-feedback/`）
- 训练营提交与 PoW 材料（`submissions/`）
- 黑客松方向草稿（`hackathon/`）

## 目录说明

```text
README.md              # 本文件
profile.md             # 学员画像（Learning Agent 维护）
learning-plan.md       # 个人学习计划
agent-config.md        # Learning Agent 工具配置（不含密钥）
daily/                 # 每日学习与打卡记录
tasks/                 # 按周/模块拆分的任务笔记
experiments/           # AI × Web3 交叉实验
handbook-feedback/     # 对 Handbook 的反馈（可公开、可索引）
hackathon/             # 黑客松想法与原型记录
submissions/           # 对外提交用的整理版材料
templates/             # 笔记与打卡模板
```

## 隐私与安全提醒

本仓库默认 **public**。请勿提交：

- API Key、GitHub Token、WCB Secret API Key
- 钱包助记词、私钥、keystore
- 未公开联系方式、内部会议链接、他人个人数据

链上实验只记录：**测试网地址、交易哈希、合约地址、区块浏览器链接**。
