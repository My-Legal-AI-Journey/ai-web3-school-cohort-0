## 学习策略（针对法律方向 + vibe coding）

1. **AI 模块**：用概念卡片 / 流程图 / quiz 代替深写代码；术语对照法律场景（证据、留痕、人工复核）。
2. **Web3 模块**：优先 **Remix 图形界面 + 逐步清单**，少碰命令行；每天只完成 1 个最小动作。
3. **交叉实验**：Claude Code 出「步骤说明书」→ 你逐步核对 → 钱包点确认 → 截图/链接进 repo。
4. **PoW 标准**：笔记清晰 + 打卡完整 + 测试网链接，**不考核**独立写合约能力。
5. **安全**：助记词/私钥/API Key 永不进 repo；链上动作必须本人确认。
